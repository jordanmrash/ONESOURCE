PROJECT FILES

UNDERSTANDING PROJECT FILES

Before you create your project file, review the following so you are prepared to complete the steps in Creating Project Files (page 18)


File Extension: ONESOURCE International Tax Calculator Desktop project files are saved with the extension .itc in the location you specify each time you create a new project file.


File size: International Tax Calculator Desktop project files range in size, from just under 1 megabyte to over 20 megabytes, depending on the number of entities in your company and the amount of data loaded in International Tax Calculator Desktop.

Opening Projects: You can open International Tax Calculator Desktop project files (.itc files) only from within International Tax Calculator Desktop.

Double-clicking on an .itc file does not open a project.


Companion Files: In addition to the .itc project file, International Tax Calculator Desktop creates two other adjacent files in the location you specify:


l Backupfile(.bak):The.bakfile will grow insize asdatais addedorchanged and could range in size up to 100megabytes.


l Lock file (.lock): When you have the project file open, the .lock file prevents inadvertent editing by another user. The .lock file is very small and disappears each time a user exits International Tax Calculator Desktop.


Number of International Tax Calculator Desktop Project Files: You can create as many International Tax Calculator Desktop project files as you want. Project files in International Tax Calculator Desktop are independent of one another. You may wish to create a project file for a dry run of your 965 or GILTI calculation, create several other project files for 2018 to experiment with different calculation methods, or view planning scenarios.

Single User Access on the Hard Drive: You may save the project file either on your own local hard drive or on a shared drive. Project files saved on your own local hard drive are only accessible by you.


Multi-User Access on Shared Drives: To enable more than one user to access to your project files, save the project on a network shared drive or use a tool such as Microsoft SharePoint. International Tax Calculator Desktop does not support simultaneous multi-user access. One user at a time can access the project files and save their work to the project file on the shared drive.


When you try to access a project file that is already open by someone else, you are notified that the file is locked and are given the option to open a read-only version of the file. You can wait for a notification when the first user saves the file and exits International Tax Calculator Desktop.


IMPORTANT: File Paths and Character Limits: Remember that file paths in most software saved on a Microsoft platform are limited to 255 characters, so keep folder and file names short. Be certain that a constant connection is maintained to your network or shared drive as you work in the solution. Certain information is written to the file as you work, and if the connection is lost, you may lose your work.


Independence of Project Files: International Tax Calculator Desktop project files do not interact with each other, so the following are not automatically carried over to a new project:


l Date settings

l Data for entities


l Source Data Files (Trial balances, E&P Pools, Custom FX Rates, etc.)


l Custom FX rates

l Account or entity mappings


l Customized formatting applied to organizational charts in Chart Designer


You may reload the same source files for entities, trial balances, E&P Pools, FX rates, etc. in one project or another, and you may copy in entity and account mappings. However, customization in Chart Designer must be applied in each new file. Likewise, documents must be attached again in each new project file.


Year End: You may set different year ends for each project file you create. For example, set the year end as 12/31/2017 if you are a calendar year taxpayer, preparing a 2017 965 Calculation or 12/31/2018 if you are a calendar year taxpayer and are preparing your official 2018 TCJA calculations for FDII, GILTI etc.

The Ultimate Parent's year-end you enter here drives additional functionality throughout International Tax Calculator Desktop, so be precise. The 965 inclusion year for the CFCs in the consolidated group are determined by their year end compared to the Consolidated Parent's year end.

  3. When you are returned to the Save Project dialog box, complete the following:

OPENING PROJECT FILES

International Tax Calculator Desktop project files (.itc files) cannot be opened by double-clicking on the project file name, as you might do with a Microsoft Excel or Word file. The .itc files must be opened from within International Tax Calculator Desktop.


You can open a project file in International Tax Calculator Desktop one of the following ways:

l By selecting the project name in the Recent Projects area in the middle of your

Home screen

l By browsing to a specific file from the Project Menu in the upper right of your screen


To open an existing project:

  1. From the Project Menu in the toolbar, select the down arrow and select Load Project.

  2. In the Open File dialog box, select Browse.

  3. Locate and select the project file on your local system using the prompt.


  4. Select OK to load your project.
